## 屏幕滚动

| 按键           | 作用                   |
| -------------- | ---------------------- |
| `Ctrl`+`D`/`U` | 向下/上翻页滚屏        |
| `zz`           | 移动视窗中央到光标位置 |
| `M`            | 移动光标到视窗中央     |

## 光标移动

**vim 教条：尽量不要在插入模式下执行任何光标移动操作！**

| 按键                | 作用                                |
| ------------------- | ----------------------------------- |
| `h`\|`j`\|`k`\|`l`  | 左/下/上/右                         |
| `w`\|`e`\|`b`\|`ge` | 下个词头/下个词尾/上个词头/上个词尾 |
| `0`/`$`             | 行首/行尾                           |
| `I`/`A`             | 行首插入/行尾插入                   |
| `f`/`F` `{char}`    | 下/上个字符，`f Enter`重复上次      |
| `;`/`,`             | 选择并跳转到指定位置                |
| `%`                 | 跳转结对符                          |
| `gg`                | 第一行                              |
| `G`                 | 最后一行                            |
| `{num}` `G`         | 跳转第`{num}`行                     |

## 标签跳转

| 按键           | 作用                                  |
| -------------- | ------------------------------------- |
| `Ctrl`+`O`/`I` | 跳转上/下个位置                       |
| `g;`\|`g,`     | 跳转上/下个修改位置                   |
| `gi`           | 跳转上次插入位置                      |
| `m` `{char}`   | 设置 vim 标签                         |
| `'` `{char}`   | 跳转 vim 标签(不支持大写字母全局跳转) |
| `mm`           | 切换书签状态                          |
| `mi`           | 为书签添加注释                        |
| `ml`\|`mL`     | 列出当前/所有文件的书签               |
| `mc`\|`mC`     | 删除当前/所有文件的书签               |
| `mn`\|`mp`     | 跳转至下/上一个书签                   |
| `[c`\|`]c`     | 跳转上/下个 change                    |
| `[h`\|`]h`     | 跳转上/下个 hook                      |

## 全文搜索

| 按键                             | 作用                                  |
| -------------------------------- | ------------------------------------- |
| `/`\|`?`                         | 向下/上搜索当前文件内容               |
| `*`\|`#`                         | 向下/上搜索光标下单词, 限制单词边界   |
| `g*`\|`g#`                       | 向下/上搜索光标下单词, 不限制单词边界 |
| `:[range]s/pattern/repl/[flags]` | 替换命令                              |
| `g&`                             | 相当于`:%s/search/repl/flags`         |
| `Alt`+`C`                        | 搜索时是否大小写敏感                  |
| `Alt`+`W`                        | 搜索时是否匹配单词边界                |
| `Alt`+`R`                        | 搜索时是否使用正则引擎                |
| `Ctrl`+`F`                       | 搜索当前文件内容                      |
| `Ctrl`+`H`                       | 替换当前文件内容                      |
| `Shift`+`Ctrl`+`F`               | 搜索目录所有文件内容                  |
| `Shift`+`Ctrl`+`H`               | 替換目录所有文件內容                  |

## 普通模式

- 复合操作符： `d` `c` `gu` `gU` `g~` `y` `v`
- 文本对象： `w` `W` `(` `{` `[` `<` `"` `'` `` ` `` `e` `l` `i` `f` `,`
- 复合操作：
  - [次数] + 普通操作符
  - 复合操作符 + [次数] + 光标移动
  - 复合操作符 + 文本对象

| 按键                           | 作用                      |
| ------------------------------ | ------------------------- |
| `~`                            | 反转字符大小写            |
| `r`                            | 替换光标下字符            |
| `x`                            | 删除字符                  |
| `s`                            | 删除字符并进入 insert     |
| `R`                            | 从光标位置开始替换        |
| `D`                            | 删除直到行尾              |
| `C`                            | 删除直到行尾并进入 insert |
| `dd`                           | 删除当前行                |
| `cc`                           | 删除当前行并进入 insert   |
| `ds` `{char}`                  | 删除包围符                |
| `cs` `{old-char}` `{new-char}` | 替换包围符                |
| `ys` `{text-obj}` `{char}`     | 添加包围符                |
| `J`                            | 连接下行到本行            |
| `S`                            | 断开到下行                |
| `Ctrl`+`A`/`X`                 | 递增/减光标下数字         |
| `.`                            | 重复上次修改操作          |
| `q` `{reg}`                    | 宏录制                    |
| `@` `{reg}`                    | 应用宏                    |
| `@@`                           | 应用上次宏                |

## 复制粘贴

| 按键                             | 作用                                  |
| -------------------------------- | ------------------------------------- |
| `Ctrl`+`R` `{reg}`               | 粘贴寄存器(Insert-Mode)               |
| `Ctrl`+`V`                       | 粘贴系统剪切板(Insert-Mode)           |
| `p`\|`P`                         | 粘贴 " 号寄存器到光标后/前            |
| `y`\|`Y`\|`yy`                   | 复制文本/光标后文本/本行到 0 号寄存器 |
| `=p`\|`=P`                       | 粘贴 0 号寄存器到光标后/前            |
| `=o`\|`=O`                       | 粘贴 0 号寄存器到下/上行              |
| `Space Space`                    | 将 0 号寄存器复制到系统剪切板         |
| `Space y`\|`Space Y`\|`Space yy` | 复制文本/光标后/本行到系统剪切板      |
| `Space p`/`Space P`              | 粘贴系统剪切板到光标后/前             |
| `Space o`/`Space O`              | 粘贴系统剪切板到下/上行               |
| `Space b y`                      | 复制所有内容到系统剪切板              |
| `Space b p`                      | 从系统剪切板粘贴内容覆盖当前文件      |
