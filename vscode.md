## 屏幕滚动

| 按键                | 作用                             |
| ------------------- | -------------------------------- |
| `Ctrl`+`D`/`B`      | 向下/上翻页滚屏                  |
| `z Enter`/`zz`/`zb` | 移动视窗顶部/中间/底部到光标位置 |
| `H`/`M`/`L`         | 移动光标到屏幕顶部/中间/底部     |

## 光标移动

**vim 教条：尽量不要在插入模式下执行任何光标移动操作！**

| 按键                | 作用                                |
| ------------------- | ----------------------------------- |
| `h`\|`j`\|`k`\|`l`  | 左/下/上/右                         |
| `w`\|`e`\|`b`\|`ge` | 下个词头/下个词尾/上个词头/上个词尾 |
| `0`/`Ctrl`+`E`      | 行首/行尾                           |
| `I`/`A`             | 行首插入/行尾插入                   |
| `f`/`F` `{char}`    | 下/上个字符，`f Enter`重复上次      |
| `;`/`,`             | 选择并跳转到指定字符                |
| `%`                 | 跳转结对符                          |
| `gg`                | 第一行                              |
| `G`                 | 最后一行                            |
| `{num}` `G`         | 跳转第`{num}`行                     |

## 全文搜索

| 按键               | 作用                    |
| ------------------ | ----------------------- |
| `/`\|`?`           | 向下/上搜索当前文件内容 |
| `*`\|`#`           | 向下/上搜索光标下单词   |
| `Alt`+`C`          | 搜索时是否大小写敏感    |
| `Alt`+`W`          | 搜索时是否匹配单词边界  |
| `Alt`+`R`          | 搜索时是否使用正则引擎  |
| `Ctrl`+`F`         | 搜索当前文件内容        |
| `Ctrl`+`H`         | 替换当前文件内容        |
| `Ctrl`+`Shift`+`F` | 搜索目录所有文件内容    |
| `Ctrl`+`Shift`+`H` | 替換目录所有文件內容    |

## 普通模式

- 复合操作符： `d` `c` `gu` `gU` `g~` `y` `v`
- 文本对象： `w` `W` `(` `{` `[` `<` `"` `'` `` ` `` `e` `l` `i` `f` `,`
- 复合操作：
  - [次数] + 普通操作符
  - 复合操作符 + [次数] + 光标移动
  - 复合操作符 + 文本对象

| 按键                           | 作用                      |
| ------------------------------ | ------------------------- |
| `~`                            | 反转字符大小写            |
| `r`                            | 替换光标下字符            |
| `x`                            | 删除字符                  |
| `s`                            | 删除字符并进入 insert     |
| `R`                            | 从光标位置开始替换        |
| `D`                            | 删除直到行尾              |
| `C`                            | 删除直到行尾并进入 insert |
| `dd`                           | 删除当前行                |
| `cc`                           | 删除当前行并进入 insert   |
| `ds` `{char}`                  | 删除包围符                |
| `cs` `{old-char}` `{new-char}` | 替换包围符                |
| `ys` `{text-obj}` `{char}`     | 添加包围符                |
| `J`                            | 连接下行到本行            |
| `Ctrl`+`A`/`X`                 | 递增/减光标下数字         |
| `.`                            | 重复上次修改操作          |
| `q` `{reg}`                    | 宏录制                    |
| `@` `{reg}`                    | 应用宏                    |
| `@@`                           | 应用上次宏                |

## 复制粘贴

| 按键               | 作用                             |
| ------------------ | -------------------------------- |
| `Ctrl`+`R` `{reg}` | 粘贴寄存器(Insert-Mode)          |
| `p`\|`P`           | 粘贴 " 号寄存器到光标后/前       |
| `y`                | 复制文本到 0 号寄存器            |
| `yy`               | 复制本行到 0 号寄存器            |
| `Y`                | 复制光标后文本到 0 号寄存器      |
| `=p`\|`=P`         | 粘贴 0 号寄存器到光标后/前       |
| `=o`\|`=O`         | 粘贴 0 号寄存器到下/上行         |
| `Space`+`y`        | 复制文本到系统剪切板             |
| `Space`+`Space`    | 将 0 号寄存器复制到系统剪切板    |
| `Space`+`p`/`P`    | 粘贴系统剪切板到光标后/前        |
| `Space`+`o`/`O`    | 粘贴系统剪切板到下/上行          |
| `Space b y`        | 复制所有内容到系统剪切板         |
| `Space b p`        | 从系统剪切板粘贴内容覆盖当前文件 |
